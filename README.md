## 사용법

함수 호출로 간단히 사용할 수 있습니다.

```sh
    // ink_toast
    <link rel="stylesheet" href="/ink_toast/ink_toast.css">
    <script src="/ink_toast/ink_toast.js"></script>

    // use
    i_toast("error", "Error msg");
```

더 많은 정보는 <a href="http://ink.pe.kr/">http://ink.pe.kr/</a>에서 확인할 수 있습니다.
